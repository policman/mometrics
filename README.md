# Mometrics (в разработке)
Mometrics - это backend-платформа, где пользователи создают “мониторы” своих веб-сервисов, а система в фоне регулярно проверяет указанные URL, записывает результаты, считает метрики (uptime, скорость ответа) и даёт API, через которое можно увидеть текущий статус и историю работоспособности каждого сервиса.

стек:

backend: Python 3.13, FastAPI, SQLAlchemy, Celery, Redis, PostgreSQL, Poetry,

frontend: React, TS, Vite, MUI, Recharts/Chart.js,

релиз: ~1 декабря
