# mometrics
Mometrics - это backend-платформа, где пользователи создают проекты и “мониторы” своих веб-сервисов, а система в фоне регулярно проверяет указанные URL, записывает результаты, считает метрики (uptime, скорость ответа) и даёт API, через которое можно увидеть текущий статус и историю работоспособности каждого сервиса.
